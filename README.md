# Oh My Ubuntu

Oh My Ubuntu is a script to set up a previous configured environment.
  
## Usage

```shell script
./ohmyubuntu [Target User]
```

## Software

The following software will be installed:

### Productivity

- autojump
- tig
- mlocate
